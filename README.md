#  maven-toys-sales-analysis
## Maven Toys: Strategic Sales & Inventory Analysis.
This project analyzes Maven Toys' retail data and transforms it into an intuitive business intelligence tool. The aim is to provide stakeholders with clear insights to help them understand revenue trends, product performance, and how to improve inventory at different store locations.
##Key Business Metrics (YTD)
* Total Revenue: $6.96M
* Total Profit: $1.82M
* Total Orders: 408,417
